Help Us Translate

On this page

If you’d like to help us translate Toku Budget into a language we don’t support yet, we really appreciate it. This page explains how to contribute safely and efficiently.

How it works

  1. Email us at grantmarcus1994@gmail.com with your language and region (e.g., “Portuguese – Brazil”).
  2. We’ll send a translation pack (a small ZIP) that includes:
    • Localizable.strings template with English source
    • Optional CSV if you prefer spreadsheets
    • A short style guide
  3. You translate the values and send the files back by email.
  4. We review, integrate, and ship in an update. 🎉

What to translate

We keep translations compact and focused on the in-app UI:

  • Sidebar & menu labels (Overview, Transactions, Budgets, etc.)
  • Buttons, tooltips, and status messages
  • Settings sections (Appearance, Import/Export, Sync, Trash)
  • Import CSV terms (Date, Amount, Debit, Credit, Note…)
  • Short helper text in cards (e.g., “No upcoming bills”)

Style & consistency

  • Keep it concise. Space in buttons and chips is limited.
  • Use sentence case unless your locale prefers title case.
  • Do not translate placeholders like %d, %@.
  • Match finance terms commonly used in your locale (e.g., “Debit/Credit”).
  • Dates & numbers: the app formats them for the user’s locale, so translate the labels only.

Format notes

.strings format

/* Context: Overview header */
"overview.title" = "Overview";

/* Button: starts CSV import flow */
"import.csv" = "Import CSV…";
  • Left side (key) stays English; translate only the right side (value).
  • Preserve punctuation and ellipses (…); keep any placeholders unchanged.

CSV option

Prefer spreadsheets? We can provide a two-column CSV: key, translation. Please don’t change the keys.

Testing your translation (optional)

  1. We can send you a TestFlight or a macOS build with your language added.
  2. Do a quick pass in all major sections: Overview, Transactions, Budgets, Settings.
  3. Report any truncation; we’ll tweak UI widths or wording together.

Credits

We’re grateful for community help. If you’d like, we’ll add your name (and optional link) to a translators list on the FAQ site.

Ready to help?

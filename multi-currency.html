Multi-currency

On this page

Today, Toku Budget displays amounts in a single primary currency. You’ll see USD active, and a JPY chip that’s currently locked with a “coming soon” note.

How it works today

  • The UI (Overview cards, charts) uses a primary display currency: USD.
  • Each transaction does store a currency code (e.g., USD/JPY) and it’s preserved on export.
  • In places like the Transactions table and CSV export, the per-row currency code is shown/saved.
If you import data with a Currency column, Toku Budget stores that code per transaction. Some summaries still render in the primary display currency for now.

JPY is next

You’ll notice USD and JPY chips in the top toolbar. JPY shows a lock and “coming soon”. When multi-currency ships, you’ll be able to choose a primary currency (USD/JPY) while keeping the original currency stored on each transaction.

Importing foreign amounts

When importing CSV, you can:

  • Map a Currency column (e.g., “USD”, “JPY”), or
  • Use the Currency Fallback field (defaults to USD) if your file lacks a currency column.

Amount mode:

  • Debit/Credit Columns – separate money out/in columns.
  • Single Amount – one column where negatives are expenses and positives are income.
Save your mapping as a template; it will auto-apply next time when headers match.

Planned behavior (high level)

  • Pick a primary display currency (USD/JPY to start).
  • Charts and summaries will convert to your primary currency while preserving original currency per transaction.
  • We’ll document the conversion approach and rate source when it’s released.

FAQ

Can I import JPY now?

Yes—include a Currency column with “JPY” or set the fallback to JPY. The currency code is stored, but most summaries still render in the primary display currency (USD) until multi-currency is fully available.

Can I switch the UI to JPY now?

Not yet. The JPY chip is visible but locked. When it unlocks, switching will update the UI display currency.

Will you add more currencies?

Yes—after JPY, we’ll add more. Tell us which you need most so we can prioritize.
